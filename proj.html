Warehouse Management


February 2016 â€“ current


Description:

Powered by ROS (Robot Operating System), the bot navigates autonomously performing warehouse and inventory related tasks. The idea is that the bot (Kobuki) would navigate through a room avoiding obstacles, using a map as a base, detect and lift a specified object and bring it back to the user. The algorithm for object detection is based on histogram back-projection and depth sensing using the Xbox Kinect. The robotic arm, with five degrees of freedom is controlled using the Arduino Uno microcontroller and the simultaneous localization and mapping (SLAM) is done using a LIDAR sensor. All these features will then be integrated using ROS. The bot can have promising future prospects as advanced versions of the bot can be used for Industrial Automaton and as a Butler Bot.
